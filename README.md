## Installation
* Clone the source of this library:
    ```
    git clone https://github.com/mughieams/audio-transcribe.git
    ```
* Install depedencies:
  ```
  $ pip install -r requirements.txt
  ```
* Install pyAudioAnalysis:
  ```
  cd libs\
  cd pyAudioAnalysis\
  pip install -e .
  ```
